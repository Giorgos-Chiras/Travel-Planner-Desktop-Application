# Travel Planner (WIP)

**Travel Planner** is a desktop application built using Python and the Tkinter library (with the Sun Valley Theme). This application aims to assist users in planning their trips by providing essential travel information like flights, hotels, weather forecasts, and activities. The project is still in very early development, and more features are being added regularly.

---

## Features

- **Destination & Departure Cities**: Allows users to input the destination and departure cities for travel.
- **Travel Dates**: Users can specify their travel dates.
- **Flight Search**: Fetch flight information using the Amadeus API (currently in very early development).
- **Hotel Search**: Planned feature to integrate hotel search capabilities (yet to be implemented).
- **Weather Forecast**: Planned feature to display weather forecasts (yet to be implemented).
- **Activities & POIs**: Planned feature to recommend activities and Points of Interest (POIs) (yet to be implemented).

---

## Technologies Used

- **Programming Language**: Python
- **GUI Library**: Tkinter (with the [Sun Valley Theme](https://github.com/rdbende/Sun-Valley-ttk-theme))
- **Flight Search API**: [Amadeus API](https://developers.amadeus.com/)
